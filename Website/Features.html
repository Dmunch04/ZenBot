Features

Here you'll find the full list of ZenBot's features.
This is both commands and functions.

Introduction

ZenBot is made in Python built with Discord.py. I'm using a lot of packages. Some of them are self made, others are from GitHub. You cann see all the packages in the requirements.txt file. You'll also find some about how I made it, and why I made it. If you wanna customize ZenBot and host it yourself, click the button below to get help!

Guide

Main Features

Twitter Feed

I've integrated a Twitter API, that let's you make the bot send a Discord message, each time a new tweet is posted from an optional Twitter account

Video Announcement

Settings Configuration

You can easily customize the bot to your server. From disabling/enabling commands, to switching each roles permission.

Readme

The bot is in early stages, and therefor there might be bugs and issues. We don't take responsiblity for data breaches. This means that if we get hacked and our token gets leaked, then it's not our fault if they use the token, and it harms your server. In the list below here, you'll see a list of commands. The channels and permissions for these commands are configurable!
